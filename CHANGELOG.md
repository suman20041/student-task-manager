## [2.0.0] – 2026-05

### Added
- 🎉 Complete Gamification system (XP, Levels, Coins)
- 🏅 Achievement Badges system
- 🍅 Integrated Pomodoro "Focus Mode" timer
- 📊 Advanced user statistics and streak tracking
- 💾 Data Export (CSV, PNG, PDF) with Chart.js visualization
- 🎨 Modern translucent glassmorphism UI

### Changed
- Massively refactored single-file architecture into structured gamified app
- Rebranded to "TaskQuest" to emphasize gamification of productivity

---

## [1.0.0] – 2026-03

### Added
- ➕ Add new tasks via text input
- 📋 Display task list in the browser
- 🎨 Basic CSS styling
