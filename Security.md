# 🔐 Security Policy

## 📌 Supported Versions

| Version | Supported |
| ------- | --------- |
| 1.x.x   | ✅ Supported |
| <1.0    | ❌ Not supported |

---

## 📌 Overview

Study Task Tracker is a simple web application built using HTML, CSS, and JavaScript to manage daily study tasks.

---

## 🎯 Scope

### ✅ In-Scope

- Issues that expose or manipulate user tasks unexpectedly  
- Client-side vulnerabilities affecting application behavior  
- Unauthorized modification or deletion of tasks  

---

### ❌ Out-of-Scope

- UI/UX improvements or design suggestions  
- Feature requests  
- Non-security-related bugs  

---

## 📬 Reporting a Vulnerability

If you discover a security issue, please report it responsibly.

### Please include:
- Description of the issue  
- Steps to reproduce  
- Impact of the vulnerability  
- Proof of Concept (if available)  

---

## ⏱️ Response Timeline

- Acknowledgement: Within 24 hours  
- Further updates will be shared after review  

---

## 🔒 Responsible Disclosure

- Do not publicly disclose vulnerabilities before they are resolved  
- Test only within your local environment  

---

## ⚖️ Safe Harbor

We support good-faith security research and will address valid issues responsibly.

---

## 🙌 Acknowledgements

We appreciate contributions from the community to improve this project.

If you find a security issue, please report it privately. We aim to respond within 48 hours.
